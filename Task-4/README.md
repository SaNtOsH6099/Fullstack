# Task 4 - Scenery Display

A simple web page component that displays an image card with a stylized gradient call-to-action button.

## 🚀 How to Run

1. **Download the files**: Make sure `index.html` and `style.css` are in the same folder.
2. **Add the image**: Create a folder named `images` in that same directory and place your `scenery.png` file inside it.
3. **Open the page**: Double-click `index.html` to open and view it instantly in your web browser.

## 🛠️ Features

- Centered layout using viewport units (`vh` / `vw`).
- Vibrant button with an `hsla` linear gradient.
- Clean interactive hover state for the link.
- Global `box-sizing` reset for layout consistency.
