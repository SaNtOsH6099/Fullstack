# 🧺 Laundry Service Landing Page

A sleek, modern, and fully responsive landing page designed for a premium laundry and dry-cleaning service. Built from scratch using semantic HTML5 and modern CSS3 featuring a clean layout and dynamic fluid scaling.

🔗 **Live Demo:** [https://santosh6099.github.io/Fullstack/Task-7/](https://santosh6099.github.io/Fullstack/Task-7/)

---

## ✨ Features

- **100% Fully Responsive:** Smooth transitions across desktop viewports, tablet screens, and narrow mobile devices (like the iPhone SE) using intelligent media queries and fluid flexbox grids.
- **Aesthetic Dark Theme:** A premium look blending a modern dark background with professional, eye-catching blue accents to prioritize call-to-action metrics.
- **Clean & Semantic Architecture:** Built with clean structural HTML markup and highly maintainable, component-driven CSS styling.

---

## 🛠️ Built With

- **HTML5** – Semantic document layout structure.
- **CSS3** – Custom flexbox layouts, smooth hover transitions, and responsive container break-points.

## 🚀 Getting Started (Run Locally)

Want to check out the site locally or start customizing the code? Follow these quick steps:

### Prerequisites

Make sure you have [Visual Studio Code](https://code.visualstudio.com/) installed.

### Step-by-Step Launch

1. **Download the Project:** Download or clone this repository to your local computer and open the folder inside VS Code.
2. **Install Live Server:** If you haven't already, head to the extensions marketplace tab (`Ctrl + Shift + X`), search for **Live Server** (by Ritwick Dey), and install it.
3. **Go Live:** \* Right-click anywhere directly inside your `index.html` file and select **Open with Live Server**.
   - Alternatively, click the **Go Live** icon on the bottom right of the VS Code status bar.
4. **Test Responsiveness:** Your default browser will instantly spin up a local hosting window. Hit `F12` to open DevTools, select mobile emulation, and watch the columns cleanly rearrange!
