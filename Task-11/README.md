# Thala 7 Sneaker/shoe Store (Task 11)

Designed and implemented a web application for a Sneaker/Shoe selling store . Which has a Landing Page , a Product display Page , Shopping cart view Page and a Payment Page

Click the link below to see the live deployement
[https://santosh6099.github.io/Fullstack/Task-11/](https://santosh6099.github.io/Fullstack/Task-11/)

# How to Run

1. Download all th files.
2. Make a Folder and move the downloaded files into the folder.
3. Download Live Server extension.
4. open the "index.html" file and run the live server located at the bottom right on your Screen.

# Navbar

Navigation bar made with position absolute. the styling was made seperate to be used across multiple pages.
also the nav.css includes the basic styling of the pages from background to height.

The navbar also inlcudes a hidden navbar and a hamburger button which is hidden by default with position fixed .
only visible on screens with less than 480px.
The hidden nav slides out from the right when the hamburger button is clicked.

## Features
