# 🧺 Laundry Service Landing Page (Task 9)

An enhanced, high-interaction iteration of the premium laundry service landing page. This version elevates the user experience by pairing the pure-CSS mobile navigation from Task 8 with advanced CSS3 transformations, transitions, and playful micro-interactions.

🔗 **Live Demo:** [https://santosh6099.github.io/Fullstack/Task-9/](https://santosh6099.github.io/Fullstack/Task-9/)

---

## ✨ Features

- **Dynamic Call-to-Action Micro-Interactions:** The primary action button (`.btn`) now features an engaging, high-energy hover state that scales up (`scale(1.1)`) and playfully tilts (`rotate(-10deg)`) for a tactile, modern feel.
- **Pure-CSS Interactive Mobile Menu:** Uses smart CSS `:focus-within` selectors to drive a fully functional slide-out sidebar drawer on mobile interfaces without touching a single line of JavaScript.
- **Fluidly Responsive Architecture:** Seamless multi-device adaptation that transitions elegantly across desktops, tablets, and compact mobile viewports (like the iPhone SE).
- **High-Contrast Dark Theme:** Sleek typography, subtle gradients, and glowing accents curated for an aesthetic, conversion-focused user interface.

---

## 🛠️ Built With

- **HTML5** – Clean, semantic structure optimizing document accessibility.
- **CSS3** – Advanced layouts utilizing custom flexbox grids, media queries, and performance-optimized transition states (`transform: scale() rotate()`).

## 🚀 Getting Started (Run Locally)

Want to run the code locally or experiment with more CSS transformation animations? Follow these steps:

### Prerequisites

Make sure you have [Visual Studio Code](https://code.visualstudio.com/) installed on your machine.

### Step-by-Step Launch

1. **Download the Project:** Download or clone this repository to your local drive and open the project folder inside VS Code.
2. **Install Live Server:** Head to the Extensions marketplace (`Ctrl + Shift + X`), search for **Live Server** (by Ritwick Dey), and click install.
3. **Go Live:** \* Right-click anywhere directly inside your `index.html` file and select **Open with Live Server**.
   - Alternatively, click the **Go Live** icon on the bottom-right status bar of VS Code.
4. **Test the Hover Effects:** Hover your cursor over the "Book a service today!" button to see the new dynamic scaling and rotation animation in action!
