# 🧺 Laundry Service Landing Page (Task 8)

Added a mobile-responsive sidebar navigation using purely CSS

🔗 **Live Demo:** [https://santosh6099.github.io/Fullstack/Task-8/](https://santosh6099.github.io/Fullstack/Task-8/)

---

## ✨ Features

- **Pure-CSS Interactive Mobile Menu:** Uses advanced CSS `:focus` selectors to create a fully operational slide-out navigation drawer on mobile viewports.
- **100% Fully Responsive Layout:** Seamless layout adaptation across ultra-wide desktops, tablets, and small phone interfaces .

---

## 🛠️ Built With

- **HTML5** – Modern structural markup with accessible layout elements.
- **CSS3** – Advanced layouts featuring custom media queries, state-driven animations, flexbox columns, and pseudo-class interactivity (`:focus`).

## 🚀 Getting Started (Run Locally)

### Prerequisites

Make sure you have [Visual Studio Code](https://code.visualstudio.com/) installed on your machine.

### Step-by-Step Launch

1. **Download the Project:** Download or clone this repository to your local drive and open the root directory inside VS Code.
2. **Install Live Server:** Open the Extensions tab (`Ctrl + Shift + X`), search for **Live Server** (by Ritwick Dey), and install the extension.
3. **Go Live:** \* Right-click anywhere inside the `index.html` file and select **Open with Live Server**.
   - Alternatively, click the **Go Live** button located in the bottom-right status bar of VS Code.
4. **Test the Hamburger Menu:** Open the browser's developer tools (`F12`), switch to mobile layout emulation (width under `480px`), and tap the hamburger icon (`☰`) to view the interactive pure-CSS side menu layout!
