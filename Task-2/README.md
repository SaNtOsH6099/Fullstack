# Task 2: Laundry Wallah / Laundry Mart Webpage

A simple, structured HTML webpage built for a premium laundry service. This webpage includes service lists, a pricing table, and an interactive booking form.

## 🚀 Features

- **Header & About Section:** Displays a welcoming title and a professional overview of the business.
- **Services Offered:** A clean bulleted list showing the available fabric care options.
- **Price List:** A simple HTML table highlighting specific service rates.
- **Book Now Form:** A functional input form capturing Name, Email, and Phone number.
- **Footer Contact:** Includes a direct click-to-email link using the `mailto:` protocol.

## 📂 Project Structure

- `index.html` - The webpage structure and layout.
- `laundry.png` - The placeholder image asset used in the webpage.
