## ✨ Features

- **Orbital & Anti-Rotation :** Implements `@keyframes orbit` and `@keyframes rev-shrink`
- **Squash & Stretch Physics:** Utilizes `scaleX` and `scaleY` properties ($0.7$ to $1.3$).

---

## 🛠️ Built With

- **HTML5** – Strict semantic layout markup structure.
- **CSS3 Keyframes & Vectors** – Custom timing distributions, multi-axis translations (`translateX`), physical transformations (`rotate`, `scale`), and viewport fluid responsive endpoints.

## 🚀 Getting Started (Run Locally)

Want to explore the motion paths locally or alter the timing functions of the orbital physics? Follow these steps:

### Prerequisites

Make sure you have [Visual Studio Code](https://code.visualstudio.com/) installed on your machine.

### Step-by-Step Launch

1. **Download the Project:** Pull down or clone this repository to your local architecture and load the workspace directory inside VS Code.
2. **Install Live Server:** Jump to the Extensions panel (`Ctrl + Shift + X`), search for **Live Server** (by Ritwick Dey), and tap install.
3. **Go Live:** \* Right-click anywhere within your `index.html` file and choose **Open with Live Server**.
   - Alternatively, activate the **Go Live** anchor in the bottom-right status tray of VS Code.
