# Project: Laundry Services Landing Page

A sleek, modern web landing page for laundry services built using HTML and CSS.

## Features & Responsiveness

- **Dynamic Scaling:** Built using **CSS Viewport Units** (`vw`, `vh`) to ensure typography, spacing, and layouts dynamically adapt to any screen size.
- **Fluid Layout:** Designed to look seamless and professional across mobile devices, tablets, and desktops without breaking.

---

## How to Run This Project

You can view and test this project using either of the methods below:

### Method 1: Local Setup

1. Locate the folder containing the project files on your computer.
2. Double-click the `index.html` file.
3. The page will automatically open and render in your default web browser (Chrome, Safari, Edge, Firefox, etc.).
