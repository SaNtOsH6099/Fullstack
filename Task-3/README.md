# HTML & CSS Selection Task (Task 3)

A simple web development assignment demonstrating the implementation of various CSS selectors, including element selectors, class selectors, ID selectors, and pseudo-classes, without relying on inline styles.

## 🚀 Features & Concepts Covered

- **Class Selectors:** Targeting elements using specific classes (`.head`, `.second`, `.paragraph`).
- **ID Selectors:** Applying unique styles using IDs (`#bglime`, `#para`).
- **Element Selectors:** Formatting standard HTML elements globally (`p`).
- **Advanced/Pseudo Selectors:** \* Styling specific links (`a`) globally without inline CSS or unique identifiers.
  - Targeting specific list items (`li`) dynamically inside an ordered list (e.g., changing colors of individual list items dynamically via structural pseudo-classes like `:nth-child`).

```Task-3
├── index.html   # The HTML structural layout
```
