# 🌟 Task 6 - Split-Screen Login Page

A sleek, modern, and fully responsive split-screen login interface built using clean HTML5 and semantic CSS3. The layout features a vibrant, media-focused branding panel on the left and an elegant, dark-themed authentication form on the right.

---

## 🚀 Live Demo

You can view the live deployment of this project directly in your browser:
🔗 **[Check out the Live Demo on GitHub Pages](https://santosh6099.github.io/Fullstack/Task-6/)**

---

## ✨ Features

- **Split-Screen Design:** Balanced 50-50 visual hierarchy using CSS Flexbox.
- **Modern Dark Theme:** Sophisticated form container with high-contrast elements and muted secondary links.
- **Responsive Layout:** Fluid typography and scaling using viewport units (`vw`, `vh`) for consistent sizing.
- **Smooth Micro-interactions:** Tactile `:active` button states for an enhanced user experience.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure (`<form>`, `<button>`, `<input>`).
- **CSS3:** Custom styling featuring Flexbox, viewport relative units, and reset practices.

## 💻 How to Run This Project Locally

1. **Download** or clone this repository to your local machine.
2. Navigate into the project folder.
3. Double-click the `index.html` file, or right-click it and choose **Open With** and select your preferred web browser (Chrome, Edge, Safari, Firefox).
